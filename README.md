# «Фокус» — менеджер списка дел

Многопользовательское веб-приложение на Python для создания и контроля задач. Данные сохраняются в SQLite, а каждый пользователь видит только собственный список.

## Возможности

- регистрация, вход и безопасный выход;
- отдельный список для каждого пользователя;
- создание, изменение, завершение и удаление задач;
- срок, описание и три уровня приоритета;
- поиск и фильтры по статусу и приоритету;
- статистика активных, выполненных и просроченных задач;
- адаптивный интерфейс и тёмная тема;
- хеширование паролей, проверка владельца и параметризованные SQL-запросы;
- 8 автоматических тестов.

## Быстрый запуск

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
python run.py
```

Откройте `http://127.0.0.1:5000` и зарегистрируйтесь. База создастся в `instance/todo.sqlite3`. При обновлении старой версии существующие задачи получит первый зарегистрированный пользователь.

## Тестирование

```powershell
python -m pytest -q
```

## Структура

```text

├── run.py                 # точка запуска
├── requirements.txt       # зависимости

│   ├── __init__.py        # фабрика приложения

│   ├── database.py        # SQLite и миграция
│   ├── models.py          # логика личных задач

│   ├── schema.sql         # таблицы и индексы
│   ├── templates/         # HTML-шаблоны
│   └── static/            # CSS и JavaScript
├── tests/                 # автоматические тесты
└── docs/                  # документация
```

## Архитектура и безопасность

Фабрика `create_app` собирает конфигурацию и два Blueprint. `auth.py` управляет аккаунтами и сессиями, `routes.py` — запросами задач, `models.py` — проверками и запросами к данным, `database.py` — соединением и обновлением схемы. Пароли сохраняются только как хеши Werkzeug. Каждая операция с задачей фильтруется по `user_id`.

Для публикации задайте случайный секретный ключ:

```powershell
$env:SECRET_KEY = "длинная-случайная-строка"
python run.py
```
